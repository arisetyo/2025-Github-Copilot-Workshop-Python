# README

## About

We are using this repository for Github Copilot workshop 2025.

## Features

This repository includes a Kitchen Game Manager with the following optional enhancements:

### 1. User Authentication
- User registration and login system
- Secure password hashing
- Personalized progress tracking across sessions
- Session management

### 2. Advanced Analytics & Dashboard
- Game statistics tracking (time played, games completed)
- Recipe performance metrics (success rate, completion rate)
- Visual dashboard display
- Leaderboard system ranking top players

### 3. Internationalization (i18n)
- Multi-language support (English and Japanese)
- Runtime language switching
- Translation file management
- Easy to extend with additional languages

## Quick Start

### Run the Enhanced Application
```bash
python3 main.py
```

### Run Individual Module Demos
```bash
# Test user authentication
python3 user_auth.py

# Test analytics system
python3 analytics.py

# Test internationalization
python3 i18n.py

# Run the original kitchen game
python3 deliverManager.py
```

### Run Tests
```bash
python3 test_enhancements.py
```
